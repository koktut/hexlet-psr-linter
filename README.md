## Requirements

* PHP >= 7.0
* Composer

## Installation

```
git clone https://github.com/koktut/hexlet-psr-linter.git
cd hexlet-psr-linter
make install
```

## Test
```
make test
```

## Usage

```
./scripts/psrlint path_to_php_file
```
